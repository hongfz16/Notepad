◆ OnSize()

void CMainFrame::OnSize ( UINT  nType,
int  cx,
int  cy 
)

当窗口大小被调整之后调用的消息响应函数
当窗口大小被调整,共有两个地方需要调整

  • 子窗口的大小
    通过调用UpdateClientRect函数
    参见
    UpdateClientRect
  • 滚动条的一系列调整
    滚动条的长度调整
    滚动条上拖动块的位置调整
    参数
    [in]nType参阅MSDN官方文档
    [in]cx由系统传入当前窗口x方向的长度
    [in]cy有系统传入当前窗口y方向的长度
    注解
    无需自己调用,当手动改变窗口大小的时候,系统会传入ON_WM_SIZE消息,此时会自动调用该函数
