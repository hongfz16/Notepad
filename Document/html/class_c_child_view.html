◆ OnPaint()

void CChildView::OnPaint ( )
protected

绘图函数
当程序接受到ON_WM_PAINT的消息的时候就会调用该函数
该函数所做工作包括

  • 重新计算所有文字的位置
  • 重新绘制画面，包括所有文字以及背景色
  • 将需要重绘的flag(即text_changed_f)置为flase
    Note
    请不要手动调用该函数，如需重绘画面请用Invalidate(false);函数
